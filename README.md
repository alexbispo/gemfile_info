# GemfileInfo

This gem shows a description about each gem in your Gemfile.

It's good to know about gems in our Rails app.

## Installation

Add this line to your Rails application's Gemfile:

```ruby
gem 'gemfile_info'
```

And then execute:

    $ bundle

## Usage

    $ rails gemfile_info:desc
